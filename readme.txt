This plugin manages job applications and allows people to submit applications.
 
== Description ==
 
This plugin allows people to submit job applications. It creates two pages/menus called 'Find a Job' and 'Find Talent'. 
You need to add positions and at least one agent in order for this two pages to be created and for user to be able to submit applications.
People can submit job applications in 'Find a Job' tab, that other users can see in a tab called 'Find Talent'. 
 
== Installation ==
 
1. Install and activate the plugin (if your execution time is set to 30 sec you will need to acitvate it twice)
2. Under admin menu named 'HR Setup' in 'Add positions' add a job position for the first files to be created
3. You also need to add at least one agent (example: WEBSITE, Your Country) in order for people to submit an application
